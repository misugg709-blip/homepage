


# (사)미래이룸학술협회 홈페이지 제작 지시서

> 이 문서는 Claude Code에서 그대로 실행할 수 있는 **홈페이지 빌드 스펙**입니다.
> 참고 사이트: `https://www.kcea.pe.kr` (한국 평생교육·민간자격증 교육 협회)
> 아래 내용을 순서대로 구현하면 유사한 구조의 협회 홈페이지가 완성됩니다.

---

## 0. 실행 방법 (Claude Code)

```
# 이 md 파일이 있는 폴더에서 실행
claude "미래이룸학술협회_홈페이지_제작지시서.md 파일을 읽고, 그 안의 스펙대로 홈페이지를 만들어줘"
```

- 결과물은 별도 빌드 도구 없이 브라우저에서 바로 열리는 **정적 사이트**로 만든다.
- 모든 CSS/JS는 각 HTML 파일 또는 공통 `assets/` 폴더로 관리한다.
- 실제 이미지는 없으므로 `https://placehold.co/{width}x{height}` 형태의 플레이스홀더를 사용한다.

## 2. 기술 스택

- **HTML5 + CSS3 + Vanilla JavaScript** (프레임워크 없이 가볍게)
- 선택: Tailwind CDN을 써도 되지만





, 유지보수를 위해 **순수 CSS(공통 style.css)** 권장
- 폰트: Pretendard (`https://cdn.jsdelivr.net/gh/orioncactus/pretendard/dist/web/static/pretendard.css`)
- 아이콘: 이모지 또는 인라인 SVG (외부 의존 최소화)
- 슬라이더/캐러셀: 순수 JS로 간단 구현 (라이브러리 불필요)

## 3. 디자인 토큰

```css
:root {
  /* 참고 사이트와 유사한 신뢰감 있는 파란/청록 계열 */
  --color-primary: #04668B;      /* 메인 파랑 */
  --color-primary-dark: #034e6b;
  --color-accent: #F5A623;       /* 포인트 주황 (버튼·강조) */
  --color-bg: #ffffff;
  --color-bg-soft: #f5f8fa;
  --color-text: #1f2d3d;
  --color-text-soft: #5b6b7b;
  --color-border: #e4ebf0;
  --radius: 12px;
  --shadow: 0 4px 16px rgba(4,102,139,0.08);
```

- 전체 톤: 깔끔하고 신뢰감 있는 교육기관 느낌
- 강조 버튼(수강신청·자격증발급)은 주황 포인트로 눈에 띄게
- 여백을 충분히 두고 카드형 레이아웃 사용

## 4. 사이트 구조 (파일/폴더)

```
project/
├── index.html            # 메인 홈페이지
├── lecture.html          # 강의 목록
├── lecture-detail.html   # 강의 상세 (참고 사이트 view 페이지에 해당)
├── certificate.html      # 자격증 안내·발급
├── about.html            # 협회 소개
├── community.html        # 커뮤니티(공지/문의/후기)
└── assets/
    ├── css/style.css     # 공통 스타일
    ├── js/main.js        # 공통 스크립트(메뉴, 슬라이더 등)
    └── img/              # 로고·플레이스홀더 등
```

> 우선 `index.html`부터 완성하고, 나머지 페이지는 동일한 헤더/푸터를 공유하도록 만든다.

## 5. 공통 요소 (모든 페이지)

### 5-1. 상단 헤더 (GNB)
- 좌측: 로고 "미래이룸학술협회" (텍스트 + 심볼)
- 우측 유틸: `로그인` / `회원가입` / `자격증 발급조회` / 검색 아이콘
- 메인 내비게이션 메뉴:
  1. **강의** (드롭다운: 아래 카테고리 목록)
  2. **자격과정 안내**
  3. **자격증 발급**
  4. **커뮤니티** (공지사항 / 1:1문의 / 수강후기 / 이벤트)
  5. **협회소개**
- 모바일: 햄버거 메뉴 → 슬라이드 사이드바
- 상단 고정(sticky) 헤더

### 5-2. 하단 푸터
- 협회 정보: 상호 `(사)미래이룸학술협회`, 대표자명(예시), 사업자등록번호(예시), 주소(예시), 대표전화 `1600-0000`, 이메일
- 링크: 이용약관 · 개인정보처리방침 · 오시는 길
- 고객센터 안내 (평일 09:00~18:00, 점심 12:00~13:00)
- 카카오톡 상담 버튼
- 저작권: `© 2026 미래이룸학술협회. All rights reserved.`

### 5-3. 플로팅 버튼 (우측 하단 고정)
- 카카오톡 상담 / 전화문의 / TOP 이동

## 6. 메인 페이지(index.html) 섹션 순서

참고 사이트 구조를 반영하되, 아래 순서로 배치한다.

### (1) 히어로 배너 (슬라이더)
- 3장 자동 슬라이드 캐러셀
- 문구 예시:
  - "국가와 함께하는 평생교육, 미래이룸학술협회"
  - "100% 온라인 수강 · 자격증까지 한 번에"
  - "PC와 모바일 어디서나 자유롭게 학습하세요"
- 각 슬라이드에 CTA 버튼: `강의 둘러보기`, `자격증 안내`

### (2) 핵심 안내 4단계 프로세스
`회원가입 → 수강신청 → 온라인 학습 → 시험/수료 → 자격증 발급`
- 아이콘 + 짧은 설명이 있는 가로 스텝 카드

### (3) 인기 강의 (카드 그리드/캐러셀)
- 상위 8개 강의 카드
- 카드 구성: 썸네일(placehold.co), 카테고리 뱃지, 강의명, 총 강의 수(예: 34강), "수강신청" 버튼
- 예시 강의명(플레이스홀더):
  - 아동심리상담사 2급
  - 노인심리상담사
  - 반려동물관리사
  - 방과후지도사
  - 심리분석사
  - 미술심리상담사
  - 요양보호사 실무
  - 바리스타 자격과정

### (4) 자격증 카테고리 그리드
- 아이콘 카드형 12~20개 카테고리
- 예시: 심리/상담, 노인/요양, 아동/보육, 반려동물, 교육/지도, 환경, 미용/뷰티, 커피/제과, 예술/공예, 직업/창업, 의료/건강, 자기계발
- 클릭 시 `lecture.html?cat=...` 로 이동

### (5) 협회 소개 배너
- 좌: 소개 문구 + "협회 더 알아보기" 버튼
- 우: 협회 이미지(placehold.co)
- 강조 문구: "미래이룸학술협회는 체계적인 평생교육 커리큘럼으로 여러분의 성장을 지원합니다."

### (6) 공지사항 / 이벤트 탭
- 최신 공지 5건, 이벤트 배너 영역
- 예시 공지: "2026년 상반기 자격 검정 일정 안내", "신규 강의 오픈 안내"

### (7) 수강 후기 (슬라이더)
- 별점 + 짧은 후기 카드 3~4개 (예시 텍스트)

### (8) 하단 신뢰 배지
- "민간자격 등록번호: 2026-000000(예시)", "누적 수강생 00,000명", "강의 만족도 98%" 등 통계 스트립

## 7. 강의 상세 페이지(lecture-detail.html)

참고 사이트의 `lecture/view.asp` 페이지에 해당. 구성:

- 상단: 강의명, 카테고리, 대표 이미지, `수강신청` 버튼(주황 강조)
- 강의 정보 표: 총 강의 수, 총 수강 시간, 수료 기준(예: 진도율 60% 이상 + 시험 60점 이상), 수강 기간
- 탭 영역:
  1. **강의 소개** (커리큘럼 소개 문구)
  2. **커리큘럼 목록** (예: 1강~34강, 각 강 제목 + 재생시간 21~27분)
  3. **자격증 정보** (발급 자격증명, 등급, 활용 분야)
  4. **환불 규정 / 수강 안내**
- 하단: 연관 추천 강의 4개 카드

## 8. 나머지 페이지 요약

- **lecture.html**: 좌측 카테고리 필터 + 우측 강의 카드 그리드(정렬: 인기순/최신순), 페이지네이션
- **certificate.html**: 자격증 종류 목록, 발급 절차 안내, 발급조회 폼(이름+생년월일 입력 UI만), 자격증 샘플 이미지
- **about.html**: 협회 인사말, 설립 목적, 연혁(타임라인), 조직 안내, 오시는 길(지도 자리 placehold)
- **community.html**: 공지사항 리스트, 1:1 문의 폼, 수강후기 리스트

## 9. 반응형 기준

- 브레이크포인트: `1024px`(태블릿), `768px`(모바일)
- PC: 카드 4열 → 태블릿 2열 → 모바일 1열
- 모바일에서 GNB는 햄버거 메뉴로 전환
- 터치 슬라이더 지원(간단한 swipe)

## 11. 주의사항

- 참고 사이트의 **문구·이미지·로고를 그대로 복제하지 말 것**. 구조/레이아웃만 참고하고 콘텐츠는 위 예시(플레이스홀더)로 채운다.
- 실제 협회 정보(대표자, 사업자번호, 주소, 자격증 등록번호, 강의명 등)는 추후 교체할 수 있도록 **한 곳에 모아 주석 처리**하거나 상수로 관리한다.
- 전화번호·등록번호 등은 모두 `(예시)` 표기 후 실제 값으로 교체 안내.
