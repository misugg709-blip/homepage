# 🎱 천운 로또

로또당첨프로그램을 비교·추천하는 정적 웹사이트입니다.

## 주요 기능
- 🏆 **프로그램 추천 목록** — 별점·가격·기능 비교, 무료/유료/AI 필터
- 🔢 **로또 번호 생성기** — 완전 랜덤 / 통계 가중치 모드, 다중 세트 생성
- 📊 **번호별 출현 통계** — 인터랙티브 막대 차트
- ❓ **FAQ 아코디언**
- 🌙 **다크모드** (설정 저장) · 반응형 디자인

## 실행 방법
설치 없이 `index.html`을 브라우저로 열면 됩니다. 또는 로컬 서버:

```bash
python3 -m http.server 8000
# http://localhost:8000 접속
```

## 파일 구성
- `index.html` — 페이지 구조
- `style.css` — 디자인 (라이트/다크 테마)
- `script.js` — 데이터 및 인터랙션

> ⚠️ 정보 제공용 데모입니다. 어떤 프로그램도 당첨을 보장하지 않습니다.
