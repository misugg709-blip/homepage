# (사)미래교육이룸학술협회 홈페이지

기업 사회공헌(CSR) 기부금으로 교육비를 지원하여 평생교육·민간자격 과정을 제공하는 협회 홈페이지입니다.
빌드 도구 없이 **HTML + CSS + Vanilla JS**로 만든 정적 사이트입니다.

> ⚠️ **현재 모든 콘텐츠는 예시(placeholder) 데이터입니다.** 오픈 전 반드시 실제 값으로 교체하세요. (아래 [교체 가이드](#교체-가이드) 참고)

## 사이트 성격

협회는 수강료로 운영되는 사설 교육기관이 아니라, **기업 사회공헌부의 기부금을 교육으로 전환하는 비영리 학술협회**입니다.
따라서 사이트는 두 갈래 방문자를 모두 받도록 설계했습니다.

| 대상 | 진입 동선 |
|------|-----------|
| 학습자 | 히어로 → 인기 교육과정 → 분야별 과정 → 교육신청 (후원 과정은 교육비 0원) |
| 기업 사회공헌 담당자 | 상단 유틸 "기업 사회공헌 제휴문의" · GNB "사회공헌" → `csr.html` → 제휴 문의 |

교육 운영은 계열 교육기관인 **미래인재교육원**과 공동 수행하는 것으로 표기했습니다.

## 페이지 구성

| 파일 | 내용 |
|------|------|
| `index.html` | 메인 — 히어로 슬라이더, 5단계 프로세스, 사회공헌 모델, 인기과정, 분야별 탭, 카테고리, 후원기업, 임팩트, 협회소개, 공지/소식/일정, 후기, CSR CTA, 통계 |
| `csr.html` | 사회공헌 — 운영 모델, 후원기업, 참여 방식 4종, 후원 혜택, 사업 사례, 협약 절차, 제휴 문의 폼 |
| `lecture.html` | 과정 목록 — 분야 필터 + 교육비 지원 필터 + 정렬 + 페이지네이션 (URL 동기화) |
| `lecture-detail.html` | 과정 상세 — `?id=N`, 강의정보표, 4개 탭(소개/커리큘럼/자격증/환불), 연관 추천 |
| `certificate.html` | 자격과정 — 발급 절차, 발급조회 폼, 자격증 샘플, 자격증 목록, 초보자 가이드 |
| `about.html` | 협회소개 — 인사말, 설립 목적, 연혁 타임라인, 조직도, 오시는 길 |
| `community.html` | 커뮤니티 — 공지 게시판, 이벤트, 수강후기, 1:1 문의 폼 |
| `lotto/` | 기존 천운 로또 사이트 (별도 보존) |

공통 헤더·푸터·플로팅 버튼은 `assets/js/main.js`가 각 페이지에 렌더링합니다.

## 실행 방법

```bash
python3 -m http.server 8080
# http://localhost:8080 접속
```

`index.html`을 브라우저로 바로 열어도 동작합니다.

## 교체 가이드

실제 정보는 **`assets/js/main.js` 상단 상수에 모여 있습니다.** HTML을 고칠 필요 없이 여기만 바꾸면 전 페이지에 반영됩니다.

| 상수 | 교체 대상 |
|------|-----------|
| `SITE` | 협회명, 대표자, 사업자번호, 통신판매업신고, 주소, 전화, 이메일, 민간자격 등록번호, 카카오톡 URL |
| `SITE.csr` | 사회공헌 전용 회선·이메일·담당자, 기부금단체 지정번호, 세제 혜택 문구 |
| `STATS` | 수혜자 수, 후원 기업 수, 지원 규모, 만족도 |
| `PARTNERS` | 후원 기업명·후원 시작연도·지원 분야 |
| `IMPACT_CASES` | 사업 사례 (후원사, 사업명, 기간, 인원, 설명) |
| `CATEGORIES` | 자격증 분야 12종 |
| `LECTURES` | 과정명, 분야, 강의 수, 교육비, 후원 구분(`full`/`partial`/`self`), 후원사, 자격증명 |
| `NOTICES` / `NEWS` / `SCHEDULES` / `EVENTS` / `REVIEWS` | 공지·소식·일정·이벤트·후기 |
| `thumbOf()` | 강의 썸네일 — 실제 이미지 경로로 교체하면 전체 반영 |

HTML에도 일부 예시값이 하드코딩되어 있습니다. 오픈 전 확인:

```bash
grep -rn "(예시)" *.html assets/js/main.js
```

### 교육비 후원 구분

`LECTURES`의 `fund` 값이 카드·상세의 금액 표기를 결정합니다.

- `full` — 기업 후원 100% 지원, 수강생 부담 **0원** (발급비도 지원)
- `partial` — 일부 지원, `self` 금액만 자부담
- `self` — 후원 미배정, 교육비 전액 자부담

## 반드시 확인해야 할 법적 사항

- **후원 기업명·로고**는 현재 전부 가상(A그룹·B전자 등)입니다. 실제 기업명·로고 게시에는 **해당 기업의 사전 동의가 필수**입니다.
- **민간자격 고지 문구**(국가공인 아님, 자격기본법 등록, 취업 미보장)는 법정 고지사항이므로 삭제하지 마세요.
- **기부금 세제 혜택·기부금단체 지정번호**는 실제 지정 여부를 확인한 뒤 표기하세요.
- 통계·연혁·사업 사례는 모두 예시이므로 실제 데이터로 교체 전에는 대외 공개하지 마세요.

## 미연동 기능 (프런트 UI만 구현)

로그인·회원가입·검색·수강신청·카카오톡 상담과 모든 폼(제휴 문의, 1:1 문의, 발급조회)은 UI만 있고 서버 연동이 없습니다.
클릭 시 데모 안내 alert이 뜹니다 — `main.js`의 `data-demo` 속성과 각 폼의 `submit` 핸들러를 실제 API로 교체하세요.
